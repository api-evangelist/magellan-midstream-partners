# Magellan Midstream Partners (magellan-midstream-partners)

Magellan Midstream Partners owned the longest refined petroleum products pipeline system in the United States, along with refined petroleum products and crude oil storage facilities. Magellan Midstream Partners was acquired by ONEOK in September 2023 and its assets and operations have been integrated into ONEOK's Refined Products and Crude segment.
